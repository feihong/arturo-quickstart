# Feihong's Arturo quickstart

# Installation 

    curl -sSL https://get.arturo-lang.io | sh

Add `export PATH=~/.arturo/bin:$PATH` to your `~/.bash_profile`.

To uninstall, `rm -rf ~/.arturo`.

## Commands

Start REPL

    arturo
